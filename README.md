# AJAX Dogs

A simple AJAX demo from Frontend Masters. Fetches a random dog image from the [Dog CEO API](https://dog.ceo/dog-api/) on each button click and appends it to the page.

## Technologies

- HTML5
- JavaScript (Fetch API)
